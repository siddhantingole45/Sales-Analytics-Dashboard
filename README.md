# 📊 End-to-End Sales Analytics Dashboard | MySQL + Tableau

## 🚀 Project Overview
This project presents a complete **end-to-end sales analytics solution** built using **MySQL** for data processing and **Tableau** for visualization.
It analyzes sales performance trends and provides **actionable business insights** to support strategic decision-making for sales managers and executives.

## 🧠 Problem Statement
Organizations need a clear understanding of:
* Sales performance over time
* Customer behavior and engagement
* Profitability across products and regions

This project addresses these needs through **interactive dashboards** that enable deep data exploration.

## 🛠️ Tech Stack
* **Database:** MySQL
* **Visualization Tool:** Tableau
* **Data Processing:** SQL (Joins, Aggregations, Filtering)

---

## 📌 Dashboards Developed

### 1️⃣ Sales Dashboard

#### 🎯 Purpose
To provide an overview of sales metrics and analyze **year-over-year (YoY) performance trends**.

#### 📊 Key Features

* **KPI Overview**

  * Total Sales (Current vs Previous Year)
  * Total Profit
  * Total Quantity Sold

* **Sales Trends**

  * Monthly breakdown for current & previous year
  * Highlights highest and lowest performing months

* **Product Subcategory Analysis**

  * Comparison of sales across subcategories
  * Sales vs Profit comparison

* **Weekly Trends**

  * Weekly Sales & Profit visualization
  * Average weekly performance
  * Highlight above/below average weeks

---

### 2️⃣ Customer Dashboard

#### 🎯 Purpose

To analyze **customer behavior, segmentation, and engagement trends**.

#### 📊 Key Features

* **KPI Overview**

  * Total Customers
  * Sales per Customer
  * Total Orders

* **Customer Trends**

  * Monthly trends for all KPIs
  * Identification of peak and low-performing months

* **Customer Distribution**

  * Segmentation based on number of orders
  * Insights into loyalty and engagement

* **Top 10 Customers**

  * Ranked by profit contribution
  * Includes:

    * Rank
    * Number of Orders
    * Sales
    * Profit
    * Last Order Date

3. Connect Tableau to MySQL database
4. Open the Tableau dashboard file (.twbx)
5. Interact with filters and explore insights
